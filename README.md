# Enterprise AI Decision Intelligence Copilot

A production-grade enterprise AI platform that enables organisations to upload datasets, generate insights, build predictive models, interact with enterprise knowledge through RAG, and automate workflows using AI agents.

## Features

- Multi-tenant architecture
- RBAC (Role-Based Access Control)
- JWT Authentication
- Dataset Management
- Analytics & Reporting
- Machine Learning Workflows
- Retrieval-Augmented Generation (RAG)
- AI Agents

## Tech Stack

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT Authentication

### Frontend
- React
- TypeScript
